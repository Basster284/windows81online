# Каждый файл

### Все файлы в корневой директории

#### `app.py`
Самый главный файл, в котором развёртывается приложение.

**Функции**:

`mainpage()`: Возвращает главную страницу сайта, не принимает ни каких аргументов. (`mainpage.html`)

`startmenu()`: Возвращает страницу меню пуск, не принимает никаких аргументов. (`startmenu.html`)

`geodeshop()`: Имеет два исхода: без параметров и с параметрами, не принимает никаких аргументов. (`geodestore.html` и `geodesearch.html`)\
**Без параметров:**
Возвращает страницу загрузки магазина (`geodestore.html`)
**Параметр _q_:**
Возвращает страницу поиска (`geodesearch.html`)

`checkcon()`: Возвращает код-статус полученный с индекса Geode, не принимает никаких аргументов.

`getmod()`: Возвращает мод с индекса Geode в формате JSON, принимает один агрумент **modid**, полученный из URL.

`textify()`: Возвращает обычный текст из Markdown в формате JSON, принимает только POST запрос, не принимает никаких аргументов.

`downloadmod()`: Возвращает мод файл, скачивает его с индекса Geode если не находится в папке `cache`, принимает один аргумент **modid**, полученный из URL. Имеет один исход: с параметрами.\
**Параметр _version_**:
Возвращает мод указанной версии в параметре _version_, если параметра нет, возвращает 422.

`getmods()`: Возвращает ответ в виде JSON с индекса Geode, не принимает никаких аргументов, все параметры перенаправляются на индекс Geode.

`geodemod()`: Возвращает страницу с модов в магазине. (`geodemod.html`)

#### `requirements.txt`

Содержит необходимые библиотеки для запуска и развёртывания приложения.

## Все файлы в директории `static/javascript`

#### `fillInput.js`

Заполняет форму поиска, беря из параметра _q_ в URL.

**Функции:**

`fillInput()`: Заполняет форму поиска, беря из параметра _q_ в URL, не принимает никаких аргументов.

**Используется в:**

`templates/geodesearch.html`.

#### `getMods.js`

Получает моды из индекса Geode через энд-поинт `/getmods`, преобразуя полученный JSON в удобный интерфейс.

**Функции:**

`getmods()`: Получает моды из индекса Geode через энд-поинт `/getmods`, преобразуя полученный JSON в удобный интерфейс, не принимает никаких аргументов.

**Используется в:**

`templates/geodestore.html`.

#### `getTime.js`

Получает время с компьютера пользователя.

**Функции:**

`gettime()`: Получает время с компьютера пользователя, не принимает никаких аргументов.

**Используется в:**

`templates/mainpage.html`.

#### `loadMod.js`

Получает мод из индекса Geode через энд-поинт `/getmods/<айди мода>`, преобразуя полученный JSON в удобный интерфейс.

**Функции:**

`loadMod()`: Получает мод из индекса Geode через энд-поинт `/getmods/<айди мода>`, преобразуя полученный JSON в удобный интерфейс, не принимает никаких аргументов.

**Используется в:**

`geodemod.html`.

#### `loadStore.js`

Проверяет соединение через энд-поинт `/checkcon`, затем посылает событие `weGotInternet`, и при успешной загрузке модов прячем загрузку.

**Функции:**

(GLOBAL) `loadFullSite()`: Проверяет соединение через энд-поинт `/checkcon`, затем посылает событие `weGotInternet`, и при успешной загрузке модов прячем загрузку, не принимает никаких аргументов.

**Используется в:**

`geodestore.html`.

#### `searchMods.js`

Ищет моды в индексе Geode через эндпоинт `/getmods`, беря параметр _q_ в URL и перенаправляя в тот же энд-поинт.

**Функции:**

`getMods()`: Ищет моды в индексе Geode через эндпоинт `/getmods`, беря параметр _q_ в URL и перенаправляя в тот же энд-поинт, не принимает никаких аргументов.

**Используется в:**

`geodesearch.html`.
